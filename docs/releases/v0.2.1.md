# v0.2.1

修复 Windows 安装版和免安装版启动时出现 `ERR_MODULE_NOT_FOUND` 的问题。

## 根因

Electron 主进程使用 Node ESM 运行，但编译产物中的相对导入缺少 `.js` 扩展。开发环境可以解析，打包后的 Node ESM 无法解析。

## 修复

- 为 Electron 主进程及其共享运行时代码补充显式 `.js` 导入扩展
- 新增打包回归测试，禁止生产运行时代码再次出现无扩展相对导入
- 实际构建并启动 Windows 打包应用验证

## 下载

- `Kimi-GUI-Setup-0.2.1-x64.exe`：Windows 安装版
- `Kimi-GUI-Portable-0.2.1-x64.exe`：Windows 免安装版
