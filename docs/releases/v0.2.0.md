# v0.2.0

Kimi GUI 的中文优先版本，并首次提供 Windows 安装包和免安装版。

## 主要变化

- 界面、配置引导、运行状态、审批、快捷任务和错误提示改为中文优先
- README 与项目截图改为中文首屏
- 新增 Windows 安装版与免安装版构建
- 新增应用品牌图标
- 新增标签触发的 GitHub Release 自动发布流程
- 修复打包后的 Electron 主进程入口和 renderer 路径

## 下载

- `Kimi-GUI-Setup-0.2.0-x64.exe`：Windows 安装版
- `Kimi-GUI-Portable-0.2.0-x64.exe`：Windows 免安装版

## 当前限制

- 真实运行时目前只接入 Kimi 提供方
- 使用真实 Kimi 会话前，仍需要本机安装并登录 Kimi CLI
- Windows 安装包暂未进行商业代码签名，首次运行可能出现 SmartScreen 提示
- macOS 与 Linux 安装包尚未提供

## 验证

- `npm run test`
- `npm run typecheck`
- `npm run lint`
- `npm run build`
- `npm run audit:publish`
- Windows 打包应用启动冒烟测试
